# 🚀 AI-Powered AIOps Intelligence Platform

This project is an AI-driven AIOps system for network monitoring, incident prediction, and anomaly detection.

## 📊 Features
- Network Monitoring Dashboard
- Incident Prediction using Machine Learning
- Real-time system metrics visualization
- Root Cause Analysis
- Interactive Streamlit UI

## 🧠 Tech Stack
- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Plotly

## 🔮 Project Idea
This system simulates an enterprise AIOps platform similar to MonetX, helping IT teams detect and predict network failures using AI.

```bash
streamlit run app.py
